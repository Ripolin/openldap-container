# Bitnami like package for OpenLDAP

## Why ?

The purpose of this image is to provide a Bitnami compatible image for OpenLDAP build with no Bitnami elements.

Instead of using OpenLDAP tar package built by Bitnami, OpenLDAP is directly checkout and compile when image is building. And a debian slim image is used to replace Bitnami minideb one.

## RTFM

To get more information about how to use and configure Bitnami OpenLDAP image, read [Bitnami documentation](https://github.com/bitnami/containers/blob/main/bitnami/openldap/README.md).
